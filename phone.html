Ska Studios Xplatform

Dave Voyles

This lab goes through the process of packaging up this template as an app store installable package for Windows Phone. The app we'll be packaging is the Ska Studios app. With these tools though, you can just as easily write an application that works for iOS, Andriod and Windows, by deploying to those platforms. It works on all of them!

By the end of this lab you will be able to:

Understand how to test an application on an Emulator
Debug an application running in an emulator
Package an application for iOS and Andriod
